# Cloud-Clipboard

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](./LICENSE)

Share your clipboard across multiple devices!

## Structure

- `apps/`
- `packages/`

## Getting started

This project use [Mise](https://github.com/jdx/mise) for managing tool versions

1. Install project tools (if you're using Mise):

   ```bash
   mise install
   ```

2. Install dependencies:

   ```bash
   pnpm install
   ```
